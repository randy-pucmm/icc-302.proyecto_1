# Evaluador de Expresiones Aritmeticas

**Proyecto 1 - Programacion Funcional en Haskell**

--

## Autores
- Randy Alexander Germosen Urena
- Hugo Fernando Concepcion Lopez

## Descripcion
Evaluador de expresiones aritmeticas usando tipos algebraicos, recursividad y manejo de errores con Maybe.

## Operaciones soportadas
Suma, resta, multiplicacion, division y negacion. Además, Maneja division por cero de forma segura.
